This is fairly strightforward.
The code is to be run on an interactive Python notebook (ipynb) like Jupyter or Google Colab. It is well commented and self explanatory.
It is not meant to be run in sequence as a Python file, as it would take ages and is not written for such a run.
Just run the cell you're interested in (after running all the cells it depends upon).

A PDF of a sample run (PDF of code.pdf) is attached for your reference, and covers all the cells
